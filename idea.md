BaatCheet is an online communication / messaging and calling Desktop App very simillar to discord. It allows friends to text chat in groups, Direct private message and voice chat in both groups and direct messages.

The app is being created out of frustration from using discord. Discord lags and has a lot of features that aren't useful for these group of friends. They need a fast and reliable platform to voice chat while gaming without losing performance, fast vc with no lag and reliable and fast text messages with image upload support. no privacy threats.

Here is the vision for the app by its potentiaal users:

- A minimal, modern and sleek dark-mode ui.
- Clerk based authentication accounts and friend adding via username. (Profile Picture support)
- Groups that have admins and mods and multiple text and voice channels exactly like discord.
- Secure private / direct messages for personal 1 to 1 conversations.
- Fast Voice chat for both direct calls and group calls.
- All basic voice chat and texting functionality like, replying, reacting, editing, deleting (text) and for vc -> Muting / deafening (Configurable keybindings for this in settings.)
- Something easy to run so no performance from gaming is taken away.

Calls and Chats should be encrypted and secure so nothing is exposed.
